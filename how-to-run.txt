# SehatSathi - How to Run

## Prerequisites
- Python 3.8+ installed
- Node.js 18+ installed
- npm or yarn package manager

## Backend Setup (FastAPI)
1. Navigate to backend directory:
   cd backend

2. Install Python dependencies:
   pip install -r requirements.txt

3. Run the FastAPI server:
   python main.py
   
   OR using uvicorn directly:
   uvicorn main:app --host 0.0.0.0 --port 8000 --reload

4. Backend will be available at: http://localhost:8000
   API docs available at: http://localhost:8000/docs

## Frontend Setup (Next.js)
1. In the root directory, install dependencies:
   npm install

2. Make sure .env.local file exists with:
   NEXT_PUBLIC_API_URL=http://localhost:8000

3. Run the development server:
   npm run dev

4. Frontend will be available at: http://localhost:3000

## Troubleshooting
- If "Error connecting to server": Make sure backend is running on port 8000
- If CORS errors: Check backend CORS settings in main.py
- If database errors: Delete sehat_sathi.db file and restart backend

## Deployment Notes
- For production, update CORS origins in backend/main.py
- Set proper SECRET_KEY in production
- Use environment variables for sensitive data
- Consider using PostgreSQL instead of SQLite for production
